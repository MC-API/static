Hello

This is mc-api.net, a collection of fast and free Minecraft related APIs.

What APIs do we offer?

There a few APIs you can use:

Function Description Path Valid response types Example
Username -> UUID Convert a username into a UUID /v3/uuid JSON or CSV (Plain Text) /example/uuid
Usernames -> UUIDs Convert multiple usernames /v3/uuid JSON or CSV (Plain Text) /example/uuid#advanced
UUID -> Username Get the Username related to a UUID /v3/name JSON or CSV (Plain Text) /example/name
Username History Get the Username history related to a UUID /v3/history JSON /example/history

A couple of rules

There are just a couple of things that aren't allowed such as:
