Documentation Favicon

We provide a simple api which allows you to retrieve the icon of a Minecraft server.

A request could look something like this to get the 'favicon' of the Hypixel server:

GEThttps://{{mcapi.region}}.mc-api.net/v3/server/favicon/mc.hypixel.net

To try this API, click the button on the right:

 PNG response will be shown here

Response Explained

The response for this endpoint will be a PNG image.

Cache information is sent in the HTTP headers, which are explained here.

If the server you queried was an SRV record we will attempt to resolve it and provide data for the original IP address and port.
